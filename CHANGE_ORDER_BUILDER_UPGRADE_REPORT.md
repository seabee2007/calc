# Change Order Builder Professional Upgrade Report

## Files Changed

- `src/pages/planner/ChangeOrderBuilderPage.tsx`
- `src/components/change-order/ChangeOrderDocument.tsx`

## Preview Improvements

- Added **Client view** / **Internal view** toggle above the preview panel (default: client).
- Preview header label updates to **Client preview** or **Internal preview** based on selection.
- Desktop preview is **sticky** (`lg:sticky lg:top-24`) with a max height and internal scroll (`lg:max-h-[calc(100vh-8rem)] lg:overflow-y-auto`).
- Mobile/tablet remain stacked without sticky behavior.
- Internal pricing summary (`ChangeOrderInternalPricingSummary`) stays in the **left form column only**, inside a dashed ÔÇťnot shown to clientÔÇŁ panel.
- `ChangeOrderDocument` receives non-persisted `ChangeOrderDocumentContext` built from project store, company settings, and live preview totals.

## Document Renderer Improvements

`ChangeOrderDocument` now includes:

- Company header (logo, name, address, phone, email, license)
- **CHANGE ORDER** box with number and date
- Project information (name, address, client, contractor)
- Change summary (title, status, reason)
- Scope and schedule impact
- **Client mode:** total change order price only (no cost/margin/markup breakdown)
- **Internal mode:** line-item tables by category plus internal pricing summary (estimated cost, gross profit, margin, markup, proposal price)
- Contract value summary (original, previous approved changes, this change order, revised)
- Standard approval statement plus optional additional terms
- Contractor and client signature blocks

## PDF Improvements

`generateChangeOrderPDF` now accepts optional `ChangeOrderDocumentContext` and renders a client-facing layout:

- White background, black text, gray section labels and bordered info boxes
- Company header and optional logo
- Right-aligned **CHANGE ORDER** box
- Project information box
- Change summary, scope, schedule impact
- Total-only pricing summary (no internal metrics)
- Contract value summary when context is provided
- Approval statement and additional terms
- Signature blocks
- Footer: **Generated by Concrete Calc** and **Page X of Y**

Builder export passes full context from saved order + project + company settings.

## Pricing Unchanged
